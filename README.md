##装置简介
本系统使用高灵敏度的CCD监控摄像头，配合UFOCapture软件进行流星监测，视频和图像文件每隔1小时同步到服务器上。文件存储地址：[meteor.bnuastro.club/file](http://meteor.bnuastro.club/file)
##目标功能
*	定时对新的视频文件进行转码，以适合网络播放
*	流星图片和视频的在线展示，进行证认和分类
*	建立流星数据库
*	对热门的流星可以提交评论、目击报告等。
